## 1. 블록 색상 시스템

### 현재 상태
- 모든 숫자 블록이 동일한 노란색/금색

### 구현 필요
각 숫자별 고유 색상 적용:

| 숫자 | 색상 | HEX 코드 (추정) |
|------|------|----------------|
| 2 | 연두색 | #90EE90 |
| 4 | 초록색 | #4ECDC4 |
| 8 | 청록색 | #45B7D1 |
| 16 | 파란색 | #5D6BE8 |
| 32 | 주황색 | #E8875D |
| 64 | 분홍색 | #E85D8C |
| 128 | 청회색 | #7BA3A8 |
| 256 | 분홍색 | #E85DA8 |
| 512 | 초록색 | #5DE87B |
| 1024 | 회색 | #888888 |
| 2048 | 주황색 (특수) | #FF8C00 |
| 4096 | 보라색 (특수) | #9B59B6 |

### 작업 파일
- `frontend/src/game/objects/Block.ts`
- `frontend/src/game/config.ts`

## 2. 특수 블록 효과 (2048, 4096)

### 현재 상태
- 일반 블록과 동일한 표시

### 구현 필요
- 2048, 4096 블록에 왕관 아이콘 표시
- 빛나는/반짝이는 효과 (sparkle/glow)
- 병합 시 특수 애니메이션

### 작업 파일
- `frontend/src/game/objects/Block.ts`

---

## 3. 콤보 시스템 UI

### 현재 상태
- 콤보 배수는 점수에 적용됨
- 시각적 표시 없음

### 구현 필요
- 콤보 발생 시 화면 중앙에 "X 콤보" 배지 팝업
- 골드 프레임 + 리본 장식
- 페이드 인/아웃 애니메이션

### 작업 파일
- `frontend/src/game/scenes/GameScene.ts`

## 4. 글로벌 랭킹 표시

### 현재 상태
- 랭킹 화면에서만 순위 확인 가능

### 구현 필요
- 게임 화면 우상단에 실시간 랭킹 표시 (#1234567)
- 지구 아이콘 + 순위 번호
- 랭크업 시 "Rank Up" 팝업 알림

### 작업 파일
- `frontend/src/game/scenes/GameScene.ts`
- `frontend/src/services/LeaderboardService.ts`

## 5. 블록 미리보기 레이아웃

### 현재 상태

### 구현 필요
- 그리드 상단 중앙에 가로로 나란히 배치
- 현재 블록: 큰 크기
- 다음 블록: 작은 크기 (우측)

### 작업 파일
- `frontend/src/game/scenes/GameScene.ts`

## 6. 상단 UI 재배치

### 현재 상태
```
[일시정지]     [점수]     [설정]
              BEST: X
```

### 구현 필요
```
[코인 430 +]  [왕관 18,404]  [지구 #102928]
              18,404
[↩️][초대]              [프리미엄][≡]
```

### 세부 항목
- [ ] 코인 표시를 좌상단으로 이동
- [ ] 코인 추가 버튼 (+)
- [ ] 최고점수에 왕관 아이콘 추가
- [ ] 글로벌 랭킹 표시 추가

## 9. 게임 모드 확장

### 🎮 2048+ 모드
- **가로 병합 추가**: 기본 모드는 세로만 병합, 2048+ 모드는 가로+세로 병합
- 더 빠른 진행, 높은 점수 가능
- 난이도: 중

### ⏱️ 타임 어택 모드
- 제한 시간 내 최고 점수 달성
- 60초 / 120초 / 180초 선택
- 별도 랭킹 보드
- 난이도: 중

### 🎯 목표 달성 모드
- "2048 만들기", "10,000점 달성" 등 미션
- 스테이지별 목표 클리어
- 보상 시스템 연동
- 난이도: 중-상

### 🆚 대전 모드 (멀티플레이)
- 1:1 실시간 대전
- 같은 블록으로 동시 진행
- 먼저 게임오버 되면 패배
- 난이도: 상 (서버 필요)
